# Movie-Recommendation-System
A content-based movie recommendation system that suggests movies similar to a given movie using text similarity techniques.
Tech Stack:

Python
Pandas
Scikit-learn
TF-IDF Vectorizer
Cosine Similarity

Workflow:

Load movie metadata
Text vectorization using TF-IDF
Compute similarity matrix
Recommend top similar movies

Use Case:
Personalized movie recommendations for streaming platforms.
